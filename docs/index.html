Core Principles

The design constraints that shape everything we build.

1

Seamless Sovereignty

Users own their data, algorithm, and identity — but never have to think about it. The agent manages sovereignty like a good accountant manages your finances.

2

Invisible Decentralization

No wallet prompts. No gas fees. No seed phrases. If the user has to understand blockchain to use the product, the product failed.

3

Protocol Over Platform

Platforms are implementations. Protocols are the language. Users move between platforms without losing data, identity, or history.

4

Agent-Mediated Everything

Agents discover services, negotiate terms, manage identity, execute payments, and curate content. The agent is the user's representative.

5

BYOA — Bring Your Own Algorithm

Full algorithmic control: weights, emotional filters, session structures, reinforcement rules. Platforms compete on content quality, not manipulation.

6

Meta-First

Content is referenced by enriched metadata, not stored centrally. The network exchanges meaning, not blobs. Full media is lazy-loaded from origin sources.

7

Value Flows to Creators

No 30% platform tax. No algorithmic demonetization. Creators publish to the protocol. Payment settles peer-to-peer with minimal transaction costs.

Stream Exchange Protocol (SEP)

An open protocol defining how stream providers and consumers exchange metadata-driven content through structured negotiation.

Enrichment Envelope

Every content item carries semantic enrichment: categories, emotional tone, energy level, cognitive load, motivation, humor, skill transfer, and session-fit indicators.

Consumer Intent

Consumers express what they want: session type, duration, category weights, hard filters, and disclosure level. The consumer always controls what the provider sees.

Provider Response

Providers return enriched payloads with confidence scores, suggested session arcs, and capability declarations. Supports payloads, errors, and redirects.

Provider Manifest

Providers self-declare their capabilities: supported exchange models, content types, guardrail policies, enrichment versions, and rate limits.

Telemetry

Optional, opt-in engagement signals. Consumers share viewing data to improve recommendations. Telemetry creates a transparent value exchange.

Three Exchange Models

Curated Payload (simple), Full Index Browse (maximum sovereignty), and Negotiated Exchange (recommended — multi-round with state tokens).
